# **App Name**: TaskZenith

## Core Features:

- Google Sign-In: Authenticate users securely via Google Sign-In.
- Task Creation: Allow users to add new tasks with descriptions.
- Task Listing: Display a list of to-do tasks, including completion status and descriptions.
- Task Updating: Enable users to edit or update existing tasks, marking them as complete or modifying descriptions.
- Task Deletion: Provide functionality to remove tasks from the list.
- Data Persistence: Store user data in Firestore to ensure tasks are saved.
- AI Powered Task Suggestion: Suggests relevant tasks to the user, incorporating user habits and existing to do items.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey a sense of focus and productivity.
- Background color: Very light Indigo (#F0F2F9), almost white, provides a clean, distraction-free surface.
- Accent color: Amber (#FFC107) to highlight interactive elements.
- Body and headline font: 'Inter', sans-serif, will be used throughout, for a modern and machined look.
- Use modern, minimalist icons from a set like Font Awesome or Material Icons for actions (add, delete, edit).
- Use a clean, card-based layout with clear visual hierarchy to organize tasks effectively.
- Subtle animations (fade-in, slide-in) when tasks are added, updated, or deleted to enhance user experience.
